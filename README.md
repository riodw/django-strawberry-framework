# Django Strawberry Framework

[![Coverage Status](https://coveralls.io/repos/github/riodw/django-strawberry-framework/badge.svg?branch=main)](https://coveralls.io/github/riodw/django-strawberry-framework?branch=main)

A DRF-inspired Django integration framework for [Strawberry GraphQL](https://github.com/strawberry-graphql/strawberry) — filtering, ordering, aggregation, and permissions.

## Status

Pre-alpha — under active development.
